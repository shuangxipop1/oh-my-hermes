# oh-my-hermes

Multi-agent orchestration system for Hermes AI.

## Quick Start

```bash
npm i -g oh-my-hermes-sisyphus@latest
omh setup
```

## Skills

| Skill | Description |
|-------|-------------|
| `/autopilot` | Full autonomous execution |
| `/ralph` | Persistence loop until completion |
| `/ultrawork` | Parallel execution engine |
| `/team` | Multi-agent coordination |
| `/plan` | Strategic planning |
| `/ultraqa` | QA cycling workflow |
| `/cancel` | Cancel active modes |

## Agents

- `architect` - System design
- `executor` - Code implementation
- `critic` - Code review
- `planner` - Task breakdown
- `debugger` - Issue diagnosis
