# Agent Catalog

## Core Agents

### architect
System design and architecture planning.

**Responsibilities:**
- Design system architecture
- Define data flows
- Evaluate trade-offs
- Identify risks

### executor
Code implementation and execution.

**Responsibilities:**
- Implement features
- Write tests
- Follow conventions
- Handle errors

### critic
Code review and quality assurance.

**Responsibilities:**
- Review code quality
- Identify bugs
- Security audit
- Suggest improvements

### planner
Task breakdown and scheduling.

**Responsibilities:**
- Create task breakdown
- Identify dependencies
- Estimate effort
- Track progress

### debugger
Issue diagnosis and fixing.

**Responsibilities:**
- Reproduce bugs
- Trace root causes
- Implement fixes
- Verify solutions

## Agent Selection

| Task Type | Recommended Agent |
|-----------|-------------------|
| Architecture | architect |
| Implementation | executor |
| Code Review | critic |
| Task Planning | planner |
| Bug Fixing | debugger |
